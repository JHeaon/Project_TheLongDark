## 시작 가이드
해당 프로젝트를 실행시키기 위해서는 아래의 명령어를 실행합니다.

```
$ git clone https://github.com/JHeaon/Project_TheLongDark.git
$ pip install -r requirements.txt
$ python manage.py runserver
```
- .env : 실행전 환경변수 파일을 설정하셔야 합니다. 

```
# Django Secret Key
SECRET_KEY=

# DB
DB_NAME=
DB_USER=
DB_PASSWORD=
DB_HOST=
DB_PORT=

# Email
SENDEMAIL=
RECVEMAIL=
PASSWORD=
SMTPPORT=
```
